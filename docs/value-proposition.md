# Value Proposition – TaskScore

## Problemstellung

Viele Studierende und junge Berufstätige haben Schwierigkeiten, ihre Aufgaben realistisch zu priorisieren.  
To-Do-Listen sind oft unstrukturiert und berücksichtigen weder den zeitlichen Aufwand noch die verfügbare Energie pro Woche.

Häufige Probleme sind:
- Zu viele Aufgaben ohne klare Gewichtung
- Unrealistische Planung
- Fehlende Motivation durch fehlendes Feedback
- Keine Übersicht über tatsächlichen Fortschritt

## Zielgruppe

Die primäre Zielgruppe von TaskScore sind:

- Studierende
- Personen mit projektbasiertem Arbeitsalltag
- Nutzer mit vielen parallelen Aufgaben

Sekundär:
- Personen, die einfache Selbstorganisation ohne komplexe Tools suchen

## Lösung durch TaskScore

TaskScore löst diese Probleme durch ein **punktbasiertes Planungssystem**:

- Jede Aufgabe erhält Punkte (Aufwand)
- Es gibt ein festes Wochenbudget
- Aufgaben können nicht unbegrenzt hinzugefügt werden
- Fortschritt wird visuell dargestellt
- Eine Belohnung-Funktion spiegelt die 
  erfolgreich  abgeschlossenen Aufgaben wieder, welche im Hauptmenü als Motivationsstütze angezeigt werden. 

Dadurch entsteht eine realistische Planung innerhalb fester Grenzen.

## Nutzen für den Nutzer

| Nutzerproblem | Lösung durch TaskScore |
|--------------|------------------------|
| Zu viele Aufgaben | Punktelimit verhindert Überplanung |
| Keine Motivation | Fortschrittsanzeige + Score |
| Keine Übersicht | Wochenplan und Fortschrittsseite |
| Unstrukturierte Listen | Tagesbasierte Planung |

## Wettbewerbsvorteil

Im Vergleich zu klassischen To-Do-Apps (z.B. Notion, Todoist):

- Kein Funktions-Overload
- Sehr einfache Logik
- Fokus auf Planung statt Features
- Lokal, transparent, nachvollziehbar

## Fazit

TaskScore bietet eine **reduzierte, realistische und motivierende** Form der Aufgabenplanung, die besonders für Lern- und Projektkontexte geeignet ist.
